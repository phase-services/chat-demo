## chat-demo
A simple real-time chat application built using React, Node.js, MongoDB and Socket.IO.

## Getting started
### back-end
Inside ./backend/ run:
```bash
$ npm i
$ node index.js
```
### front-end
Inside ./frontend/ run:
```bash
$ npm i
$ npm start
```
